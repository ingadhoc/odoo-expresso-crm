Ingreso de Datos Campañas y Planes Educativos

En la sección de configuración es posible ingresar datos referidos a Campañas y Planes Educativos, que luego se presentarán disponibles para los usuarios Corresponsales para la carga de Datos de Campaña.

 

 

Para cargar nuevas Campañas, en la opción de menú correspondiente, puede crearlas presionando el botón Crear

 

 

 

 

 

 

 

Para cargar nuevos Planes Educativos, en la opción de menú correspondiente, puede crearlos presionando el botón Crear
